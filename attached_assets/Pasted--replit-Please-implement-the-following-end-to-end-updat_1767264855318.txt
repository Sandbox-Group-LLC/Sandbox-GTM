4) FEED INTO ENGAGEMENT SIGNALS (THIS IS CRITICAL)

We need clean promotion logic similar to your meeting logic:
- Promote contacts once based on cumulative buying signals.
- Avoid inflation (no duplicate “hot leads” for repeated interactions).

A) Definitions
- A “Contact” = the canonical person record in the program context (Attendee).
- “High-Intent Audience” widget should include contacts ready for follow-up based on explicit signals (intent/outcome).

B) Scoring (simple, explainable)
Create a per-attendee cumulative score from Product Interactions:

Base points per interaction:
- Intent:
  - LOW = +1
  - MEDIUM = +3
  - HIGH = +6
- Outcome boosts:
  - Requested follow-up = +3
  - Asked for pricing = +4
  - Wants a trial/pilot = +6
  - Intro to stakeholder = +4
  - Not a fit = -10 (and optionally set a “disqualified” flag)
  - Too early = -2
- Opportunity boost:
  - $100k+ = +4
  - $50k–$100k = +3
  - $10k–$50k = +2
  - < $10k = +1

C) Promotion thresholds (opinionated defaults)
- Promote to High-Intent Audience if ANY is true:
  1) At least one interaction with intent=HIGH
  2) Cumulative score >= 10
  3) Outcome in (“Wants a trial/pilot”, “Asked for pricing”, “Requested follow-up”)

- Promote to Hot Lead if BOTH are true:
  1) High-Intent Audience criteria met
  2) AND (opportunityPotential is $50k–$100k or $100k+) OR (two+ interactions with intent >= MEDIUM)

D) Deduping
- A contact can appear only once in:
  - High-Intent Audience
  - Hot Leads
even if multiple interactions exist.
- If already promoted, do not create duplicates. Instead update “Why” explanation with cumulative signals.

E) “Why is this a Hot Lead?” drill-in
Create a computed explanation object for each promoted contact:
- Top reasons (ordered):
  - “Captured HIGH intent during Product demo”
  - “Outcome: Asked for pricing”
  - “Outcome: Wants a trial/pilot”
  - “$100k+ opportunity potential selected”
  - “Multiple high-signal interactions recorded”
- Include:
  - Total interactions count
  - Last interaction date
  - Cumulative score
  - Highest intent level seen
  - Most recent outcome
  - Opportunity bucket max

UI:
- In Engagement Signals dashboard:
  - Add a tooltip or “Why?” icon next to each promoted contact that opens a small popover showing the reasons.
  - Keep it minimalist — 3–5 bullets max.

Tooltip copy updates:
High-Intent Audience widget tooltip (update):
- “Contacts showing buying signals ready for follow-up. Contacts are promoted once based on cumulative buying signals.”

Meeting Quality tooltip (update, since we’re expanding signals overall):
- “Outcome-based intent signals captured from meetings and product interactions — built for follow-up prioritization.”

F) Data pipeline / queries
- Add aggregations for:
  - Interactions Today (program scoped)
  - Total Interactions
  - % High Intent interactions
  - Top outcomes
  - Interactions by station
- Ensure these don’t pollute attendance metrics.
